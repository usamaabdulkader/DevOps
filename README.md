# DevOps

This repository documents hands-on progression across core DevOps skill areas, beginning with Linux fundamentals and Bash scripting.

The structure is organized by focus area, with each directory containing practical exercises and structured challenges.

---

## Areas Covered

- **Linux Fundamentals** – OverTheWire Bandit walkthrough and command-line exploration.
- **Bash Scripting** – Foundational scripting exercises and a progressive, level-based challenge project.

---

## Ongoing Development

This repository will continue to evolve as I explore additional areas within the DevOps ecosystem, including:

- Containers and containerization (Docker)
- Container orchestration (Kubernetes)
- CI/CD pipeline design
- Infrastructure as Code
- Cloud platform tooling
- Monitoring and automation workflows

The focus is on building practical, hands-on competence across the DevOps stack through experimentation, problem-solving, and real-world-style exercises.
